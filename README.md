# 金铲铲阵容中控台

一个可直接打开的静态管理台页面，包含：

- 管理员登录
- 阵容库浏览
- 羁绊模拟
- 装备合成
- 阵容后台管理

说明：

- 系统仅开放这一个账户
- 不提供注册功能
- 登录态与阵容数据都保存在当前浏览器 `localStorage`

## 打开方式

直接打开同目录下的 [index.html](D:\1111aaa\index.html) 即可使用。

如果需要本地静态服务，也可以在项目目录运行：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然后访问 [http://127.0.0.1:4173](http://127.0.0.1:4173)。

## 当前技术栈

- 前端：原生 `HTML + CSS + JavaScript`
- 数据持久化：浏览器 `localStorage`
- 部署方式：静态站点

## 页面结构

### 1. 登录页

- 单账户管理员登录
- 不开放注册
- 登录后进入中控台

### 2. 阵容库

- 阵容概览指标
- 强度筛选
- 难度筛选
- 全局搜索
- 阵容详情展示

### 3. 羁绊模拟

- 选择英雄
- 自动统计羁绊人数
- 自动计算触发进度

### 4. 装备合成

- 点击两个基础散件查看成装结果
- 展示完整装备合成表

### 5. 管理后台

- 新增阵容
- 编辑阵容
- 删除阵容
- 保存到浏览器本地

## 设计方向

这版界面以更简约、更克制的深色后台为主，参考游戏资料站的信息组织方式，重点优化了：

- 信息层级
- 留白与密度平衡
- 登录与后台权限边界
- 统一的按钮、卡片、标签和表单语言
